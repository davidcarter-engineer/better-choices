# Better Choices

Better Choices is a beginner-friendly health-focused web application project.
Its goal is to help users make healthier fast food decisions.

## Project Structure

- index.html
- css/styles.css
- js/script.js
- images/

## Phase 1 Features

- Responsive homepage layout
- Header and navigation links (Home, About, Restaurants)
- Hero section with project message
- Featured Restaurants section (HTML only)
- Healthy Pick of the Day section powered by JavaScript
- Footer: Better Choices © 2026

## JavaScript Concepts Used

This project intentionally uses only JavaScript fundamentals:

- Variables
- Functions
- Math.random()
- Conditional statements (if / else)

## JavaScript Logic (Beginner Explanation)

In `js/script.js`:

1. Six variables are created to store restaurant names.
2. A function named `getHealthyPick()` is created.
3. The function generates a random number from 1 to 6.
4. Conditional statements match the random number to one restaurant.
5. The selected restaurant is saved in `healthyPick` and shown on the page.

## Styling

The site uses modern CSS with:

- Flexbox layout
- Mobile-friendly design
- Green and blue color palette
- Card-style restaurant list
- Subtle hover effects

## How to Run

1. Open the project folder in VS Code.
2. Open `index.html` in your browser.
3. Refresh the page to see a new random Healthy Pick of the Day.
